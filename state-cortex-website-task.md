# Cortex Website Transformation Task

## Context
Transform an existing creative/artistic HTML website into a landing page for **Cortex** - an Autonomous Yield Optimization Protocol on Solana.

## Current State
- **Mevcut**: Minimalist, dark-themed, audio-reactive artistic website
- **Teknoloji**: Vanilla HTML + CSS + JS with GSAP & Lenis animations
- **Dosyalar**: `index.html`, `style.css`, `animation.js`

## Hedef
Cortex protokolü için profesyonel, DeFi-odaklı landing page:
- AI agents ile yield optimization
- Solana blockchain
- Tiered staking sistemi
- Real yield dağıtımı

## Cortex Key Features
1. **Autonomous Vaults** - AI agents tarafından yönetilen vault'lar
2. **Tiered Access** - Bronze, Silver, Gold, Platinum tier sistemi
3. **Real Yield** - SOL cinsinden gerçek performans fee'leri
4. **Buyback Mechanism** - Jupiter aggregator üzerinden token buyback
5. **On-Chain Transparency** - Tüm agent kararları audit edilebilir
